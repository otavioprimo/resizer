# Image Resizer for App Store

## Install
`` npm install -g resizer-store ``

## How to use

Run on terminal
`` resizer-store ``

## Features
- Resize images
- Remove alpha channel

## Supported sizes
- IOS
  - Icon 512 x 512
  - IPhone 6.5" 1242 x 2688
  - IPhone 5.5" 1125 x 2436
  - IPad 2048 x 2732

### Android not supported yet
